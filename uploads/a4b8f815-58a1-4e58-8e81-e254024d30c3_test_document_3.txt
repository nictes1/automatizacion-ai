Este es un documento de prueba para el sistema RAG optimizado de PulpoAI.

Contenido del documento:
- El sistema ahora incluye caché de embeddings para mejorar el rendimiento
- Se ha implementado monitoreo en tiempo real del sistema
- Los embeddings se generan usando Ollama con GPU
- El sistema está optimizado para procesar archivos de manera eficiente

Información técnica:
- Modelo de embeddings: nomic-embed-text
- Caché: Redis con compresión
- Monitoreo: Métricas de CPU, memoria, GPU y servicios
- Base de datos: PostgreSQL con pgvector

Este documento servirá para probar las nuevas funcionalidades del sistema optimizado.
